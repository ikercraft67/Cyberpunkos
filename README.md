# 🌃 CyberpunkOS - Android 16 GSI

Bienvenido a la página oficial de descarga de mi Custom ROM con temática Cyberpunk.

## ⚠️ IMPORTANTE: Comprueba la compatibilidad antes de flashear

Esta ROM es una imagen GSI (Project Treble). Antes de descargar o flashear la ROM en tu móvil, conecta el dispositivo al PC con la depuración USB activada y pon este comando en tu terminal para ver si es compatible:

adb shell getprop ro.treble.enabled
